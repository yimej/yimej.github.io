TIL DEATH DO US CHART

This is a predictive model developed using machine learning to predict marital status from life events recorded by the National Longitudinal Survey of Youth 1997.

The objective of this project was to use a large data set to create and train a machine learning model.

Made with: Python, pandas, NumPy, JSON, JavaScript, D3.js, Plotly, Machine Learning

Source: National Longitudinal Survey of Youth 1997

SAMHSA QUICK STATISTICS WEB TABLES

The updated Quick Statistics website was developed while at Eagle Technologies under a contract for the Substance Abuse and Mental Health Services Administration's Behavioral Health Services Information System.

The previous Quick Statistics page was visually outdated and confusing to navigate. This enhanced version is more visually modern and utilizes JavaScript and CSS to replace confusing navigation with user-friendly interactivity to display the chosen table or document.

Made with: SAS, Python, pandas, NumPy, JavaScript, AJAX, HTML5/CSS

Source: SAMHSA

WINE REVIEW STUDY

This is a simple visual study of wine reviews scraped from WineEnthusiast found on Kaggle.

The objective of this analysis was to explore the relationships among:

  1. rating points,
  2. calculated polarity of review text,
  3. country of origin,
  4. and U.S. state of origin.

Made with: Python, pandas, vaderSentiment, JavaScript, D3.js, Plotly.js, Machine Learning

Source: Kaggle

PROJECT TYCHO CHOROPLETHS

This is a series of U.S. choropleths visualizing four infections over time from Project Tycho, a compilation of standardized global datasets of infectious diseases.

The objective of this project was to visualize geographical data over time.

Made with: Python, pandas, JavaScript, D3.js, Plotly.js

Source: Project Tycho
